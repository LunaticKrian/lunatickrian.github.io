知识导航

欢迎来到数据结构与算法学习专区 🎉 本目录是 AI Agent 后端开发 方向的面试备战笔记，以 408 考研数据结构大纲 为骨架，结合后端 / 大模型应用开发的高频考点进行扩充。

目标：既能应对八股与手撕，又能支撑工程实践中的选型决策。

学习计划 Plan​

阶段一 · 基础结构（线性表 → 树 → 图） 夯实 408 核心数据结构，掌握每种结构的「定义 / 操作 / 复杂度 / 适用场景」。

阶段二 · 核心算法（查找 / 排序 / DP） 排序是 408 大题重头戏；动态规划是后端面试必考。

阶段三 · 工程化数据结构（LRU / 跳表 / 布隆过滤器 / 字典树） 结合 AI Agent 后端场景：缓存淘汰、去重过滤、自动补全、向量检索辅助结构等。

阶段四 · 刷题与复盘 按专题刷 LeetCode Hot 100 / 剑指 Offer，每题写「思路 → 复杂度 → 易错点」三段式总结。
